# 出差报销助手项目计划书

用户授权邮箱，AI 自动识别一切出差相关的邮件和文档（机票、火车票、大巴票、公交票、网约车、酒店、油费单、发票等），提取关键信息。一次生成两种价值：出差中随时查看行程信息 + 出差后一键生成报销表。

## 目标用户
所有需要出差的打工人——从员工到老总，个人使用，不需要企业部署。

## 产品形态
Phase 1（当前）：自动化工作流验证
Phase 2（待定）：Web 应用 + 邮箱 OAuth 授权

## MVP 功能清单

### 要做
- 三大邮箱 OAuth 登录（Gmail / QQ / 163）
- 邮件扫描（最近 60 天）
- 全品类识别：机票、火车票、大巴票、公交票、网约车、酒店、油费单、发票等
- 结构化文档提取（每个品类完整结构化信息）
- 原件存档（自动下载邮件附件）
- 出差行程卡片（出差中查阅）
- 报销汇总表（出差后报销）
- Excel 导出，按类目分 Sheet
- 隐私说明页

### 不做
- 用户注册系统、移动端适配、企业对接/集成、多语言、分享/协作、连续扫描、发票验真

## 竞品
- TripIt：国际老牌，手动为主，不支持中文
- Trips Flow：2024 年发布，类似想法但 iOS only，2 年没做起来
- 每刻报销/合思商旅：企业级，需公司采购
- 携程商旅：企业商旅一体化
- 钉钉AI差旅：企业内开通

**核心差异**：现有工具都是"企业视角"，差旅助手是"个人视角"。

## 变现逻辑
Freemium：免费 3 次/月，Pro 19 元/月或 99 元/年。

## 技术栈
Next.js + Tailwind + Gmail/QQ/163 API + DeepSeek-V3 + SheetJS + Cloudflare R2/Vercel Blob + Vercel。

## 分周执行计划
- 第 1 周：环境搭建 + 邮箱接入
- 第 2 周：AI 识别 + 结构化提取
- 第 3 周：前端界面 + 出差卡片 + 报销表
- 第 4 周：打磨 + 发布

## 运营成本
约 100-200 元/月（域名 + DeepSeek API）。

## 已排除的方向
- 纯旅行场景（低频，Trips Flow 2 年没做起来）
- 内容营销/自媒体
- 中小企业自动化服务（需对接客户）
- 卖工作流模板（n8n 上 2000+ 免费模板）
- 避坑工具（法律风险）
- 旅行规划 AI（ChatGPT/豆包可替代）
- 图片上传（所有出差票据都有邮件）
